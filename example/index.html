<bb-msg-history />

A chat-style message history web component.

Basic Usage

- bbki.ng: 谁呀？ - bbki.ng: foo - bbki.ng: bar - xwy: 谁谁谁，你猴爷爷！ - bbki.ng: foo - xwy: bar 
<bb-msg-history>
  - bbki.ng: 谁呀？
  - bbki.ng: foo
  - bbki.ng: bar
  - xwy: 谁谁谁，你猴爷爷！
  - bbki.ng: foo
  - xwy: bar
</bb-msg-history>
